# Security Policy

## Supported Versions
We currently provide security updates for the following versions:


| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| 0.0.x   | :x:                |

## Reporting a Vulnerability
If you discover a security vulnerability, please do **not** open a public issue. Instead, follow these steps:

1. **Email us**: Send a detailed report to [Mathieu.engineering.company@gmail.com].
2. **Include Details**: Provide a description, steps to reproduce, and potential impact.
3. **Response Time**: We aim to acknowledge your report within 48 hours and provide a fix within 5 business days.

## Disclosure Policy
We follow a coordinated disclosure process:
1. We will validate the issue.
2. We will prepare a fix for supported versions.
3. We will notify you once the fix is ready for release.
4. We will publicly disclose the issue alongside the fix.

Thank you for helping keep this project secure!
